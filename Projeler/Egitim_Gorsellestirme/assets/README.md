# Görsel Klasörü

Buraya kendi slayt görsellerinizi koyun. `index.html` şu dosyaları bekliyor:

- `asama_1.png`
- `asama_2.png`
- `asama_3.png`
- `asama_4.png`

İstediğiniz kadar aşama ekleyebilir veya çıkarabilirsiniz; sadece `index.html` içindeki
`<section class="stage">` bloklarını ve buradaki dosya adlarını eşleştirin.
