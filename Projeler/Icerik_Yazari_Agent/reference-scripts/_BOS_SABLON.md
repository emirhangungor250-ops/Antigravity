# Referans Script Arşivi — Şablon

Bu klasör, içerik yazarı agent'inin **kendi sesini öğrenmesi için** kullandığı
referans script arşividir. Agent yeni bir script üretmeden önce buradaki
örnekleri okur ve aynı üslup/format/tonu taklit eder.

## Buraya kendi içeriğini koy

Orijinal proje, sahibinin kendi yazdığı onlarca script'i burada tutuyordu.
Paylaşım için bu içerik boşaltıldı — çünkü referans corpus tamamen sana özeldir.

**Yapman gereken:** Kendi nişin için 5-15 adet "altın standart" script yaz veya
mevcut iyi içeriklerini buraya kopyala. Agent bunları okuyup senin tarzını yakalar.

Her konsept kategorisi için ayrı bir `.md` dosyası açabilirsin. Örneğin:
- `<konsept_1>_scriptleri.md`
- `<konsept_2>_scriptleri.md`
- `ilham_scriptleri.md`  (rakip analizinden üretilmiş, doğrulanmış scriptler)

## Önerilen script formatı

```
## Script #001

### Hook
(1-2 cümle — dikkat çekici giriş)

### Script
(Ana metin — kısa paragraflar)

### Notlar
(Varsa: tablolar, kaynaklar, editör notları)
```

Format tamamen sana kalmış — agent buradaki yapıyı örnek alır. Ne kadar tutarlı
ve net olursa, agent o kadar isabetli taklit eder.
