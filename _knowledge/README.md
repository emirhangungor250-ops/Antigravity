# 🧠 Knowledge Base

Bu klasör, Antigravity ile çalışırken birikmesi gereken kalıcı bilgileri içerir.
Her yeni konuşmada bu klasördeki dosyalar bağlam sağlar; aynı şeyleri tekrar tekrar açıklamak zorunda kalmazsınız.

---

## Klasör İçeriği

| Dosya | İçerik |
|---|---|
| `profil.md` | Sizin profiliniz — kim olduğunuz, ne yaptığınız, tercihleriniz |
| `api-anahtarlari.md` | Kullandığınız tüm API anahtarları ve servisler (şablon — kendi anahtarlarınızla doldurun) |
| `calisma-kurallari.md` | Tercihleriniz, tekrarlayan talepleriniz, çalışma tarzınız |
| `hatalar-ve-cozumler.md` | Geçmişte karşılaşılan hatalar ve çözümleri |
| `estetik-tasarim-notlari.md` | Beğenilen web/görsel tasarım referansları |
| `mcp-ve-tool-optimizasyon-rehberi.md` | MCP ve araç kullanımında hız/maliyet optimizasyonu |
| `platform-checklists/` | Platform bazlı kontrol listeleri (Railway vb.) |
| `templates/` | Yeni proje başlangıç şablonları |
| `autonomous_quality/` | Otonom kalite routine'lerinin playbook'ları |
| `credentials/` | 🔐 Merkezi şifre/token deposu (git'e GİRMEZ) |

---

## Nasıl Kullanılır?

- **İlk kurulum:** `profil.md` ve `api-anahtarlari.md` dosyalarını kendi bilgilerinizle doldurun.
- **Yeni bir şey eklemek istediğinizde:** Bu dosyaları doğrudan düzenleyin ya da Antigravity'den düzenlemesini isteyin.
- **Yanlış bir bilgi varsa:** "`_knowledge/api-anahtarlari.md` dosyasındaki X bilgisini güncelle" deyin.
- **Yeni bir kural eklemek isterseniz:** `calisma-kurallari.md` dosyasına ekleyebilirsiniz.
- **Hata çözüldüğünde:** `hatalar-ve-cozumler.md` dosyasına kaydedin — aynı sorunu iki kez çözmeyin.
