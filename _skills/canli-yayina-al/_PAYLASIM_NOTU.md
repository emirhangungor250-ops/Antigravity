# Paylasim Notu — canli-yayina-al

**Mod:** A

## Ne yapildi
- `scripts/railway-token.txt` — gercek Railway token tasiyabilen dosya; pakete dahil edilmedi.
- `scripts/railway.sh` — hardcoded mutlak repo koku script konumundan turetilecek sekilde duzeltildi; kisiye ozel GitHub ornekleri `<KULLANICI>/...` yapildi.
- `SKILL.md` ve `platforms/railway.md` — kisiye ozel GitHub repo referanslari `<KULLANICI>/...` ile degistirildi.

## Ogrenci ne yapmali
- `RAILWAY_TOKEN` degiskenini ortam degiskeni veya `master.env` olarak ayarlayin.
